# Zam! Premium Feature Guardrails

Status: mixed

Cloud Sync is part of the free v1 baseline. Premium should fund expanded infrastructure and advanced planning features without paywalling core budgeting.

## Free Baseline

- Core transaction, category, budget, savings, debt, add, recent, export, and local persistence workflows remain free.
- Cloud Sync encrypted backup/sync remains available for free accounts within published limits.
- Free accounts currently include a limited number of active Cloud Sync browser slots.
- Browser-only use remains available without login, with the documented local-data-loss risk.

## Premium Direction

Premium can support:

- expanded Cloud Sync slot limits
- expanded encrypted version history
- shared budgeting
- reports
- advanced planning tools
- support and continued development

Premium must not quietly take away existing core budgeting workflows.

## Planned Candidate Areas

Cloud and security:

- expanded encrypted version history
- more active synced browser slots
- stronger local trusted-key controls
- restore/recovery tooling

Planning and automation:

- recurring transactions
- bill reminders
- forecasting
- subscription tracking
- rules-based categorization

Reports and insights:

- advanced charts
- export/report templates
- budget health summaries

Collaboration:

- shared budgets
- family/partner budgeting
- read-only advisor exports or links

## Guardrails

- Do not build Premium changes until the scope is explicit.
- Do not mix billing changes with unrelated UI polish.
- Billing changes must go through `documentation/Major-Change-Release-Gate.md`.
- Public pricing, Terms, Privacy, Trust Stack, and changelog must match actual behavior.
- Premium UI should explain what is paid without making the free tier feel broken.

## Current Pricing Reference

Premium reference price: `$3.99/month`.
